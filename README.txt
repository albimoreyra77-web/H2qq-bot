CÓMO ABRIRLO

1. Descomprimí la carpeta.
2. Abrila en Visual Studio Code.
3. Abrí la terminal.
4. Ejecutá:

npm install

5. Después:

npm run dev

6. Entrá en la dirección que muestre Vite.
